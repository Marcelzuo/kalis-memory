# 四平台 Business Profile 装修能力对比

> 2026-06-17 | 天使长 Michael 调研
>
> 「装修」定义：在平台允许范围内，按品牌意图设计页面——包括图片、视频、文案、排版、功能模块。
> 对比标准：能放什么 / 能换什么 / 能自定义到什么程度。

---

## 总览：装修自由度排名

```
自由度    平台
高 ████  Facebook      最接近「自定义品牌页」
中 ███   LinkedIn      公司页模块丰富，但排版固定
中 ██    WA Business   简版微网站（目录 + 产品卡）
低 █     Instagram     极简，只能改 Bio + Highlights + 头像
```

## 一、Facebook 品牌页 ⭐⭐⭐⭐

**装修自由度最高。实质上是一个小型 CMS——你能控制的远不止头像和封面。**

### 可自定义元素

| 元素 | 详情 | 更换频率 |
|------|------|----------|
| **头像** | 170×170px 圆形裁切 | 按需 |
| **封面图** | 820×312px（桌面）/ 640×360px（手机）· 支持视频 | 每月 |
| **用户名** | `@kalistorik` URL slug | 一次 |
| **行动按钮** | "发消息" / "WhatsApp" / "查看网站" / "立即注册" 等 15+ 类型 | 随时 |
| **简介** | 255 字符，纯文本 | 按需 |
| **故事** | 5000+ 字符，页面历史/介绍，放在「关于」标签 | 按需 |
| **置顶帖** | 任意帖置顶在时间线最上方 | 每周 |
| **模板/标签页** | 可选择「标准 / 服务 / 购物 / 餐厅」等模板，决定标签页排列顺序 | 随时 |

### 标签页模块（可自定义开关 + 排列）

| 标签页 | 内容 |
|--------|------|
| **主页** | 时间线 + 简介区 + 置顶帖 |
| **帖子** | 时间线（含照片/视频/点评三个过滤项） |
| **照片** | 自动聚合所有帖子的图片 |
| **视频** | 自动聚合所有帖子的视频 |
| **关于** | 联系方式 / 地址 / 营业时间 / 更多信息 |
| **群组** | 关联的品牌群组入口 |
| **评价** | 客户评分 + 评论（最多显示 5 星评级摘要） |
| **商店** | 添加产品目录 → 站内购物 |
| **服务** | 服务项目列表 + 价格展示（适合 sourcing 业务） |
| **活动** | 线上/线下活动页面 |

### KALIS TORIK 推荐配置

```
模板：服务 (Services)
标签页开关：
  开：主页 · 帖子 · 照片 · 视频 · 关于 · 服务 · 评价
  关：商店 · 活动 · 群组（先关，有需求再开）

行动按钮：WhatsApp → wa.me/+8613602222361
```

## 二、LinkedIn 公司页 ⭐⭐⭐

**模块化装修，排版固定但内容完全可控。B2B 能量最强。**

### 可自定义元素

| 元素 | 详情 | 更换频率 |
|------|------|----------|
| **Logo** | 300×300px | 按需 |
| **封面图** | 1128×191px | 每季度 |
| **标语** | 120 字符，Logo 下方一行 | 按需 |
| **公司简介** | 2000 字符，含链接 | 按需 |
| **公司详情** | 行业 / 规模 / 成立 / 地址 / 官网 | 一次 |
| **精选内容** | 固定 3 个帖子/LinkedIn 文章，手动挑选 | 每月 |

### 模块（固定位置，但内容自主）

| 模块 | 内容 | 能否关闭 |
|------|------|----------|
| **Home** | Logo + 标语 + 简介 + 精选内容 + 最近帖子 | ❌ 强制 |
| **About** | 简介 + 详情 + 地点 + Hashtag | ❌ 强制 |
| **Posts** | 公司页发布的所有帖子时间线 | ❌ 强制 |
| **Jobs** | 招聘岗位（不开则隐藏） | ✅ |
| **People** | 员工 LinkedIn 关联（个人号关联后出现） | ✅ |
| **Life** | 公司文化图片/视频（独立标签页） | ✅ |

### 限制

- 不能自定义标签页名称或顺序
- 不能嵌入视频到 About 区（只能放链接）
- 封面图不能是视频（仅图片）
- 没有「商店」或「服务」功能

### KALIS TORIK 推荐

```
封面图：黑底 + 居中 Logo + 金线 + "Chinese Furniture Sourcing Partner"
精选内容：最新 3 篇 Insight 文章
Life：关闭（暂无文化内容素材）
Jobs：关闭
```

## 三、WhatsApp Business ⭐⭐

**不能叫「装修」，只能叫「配置」。但功能对 B2B 即时沟通非常实用。**

### 可配置元素

| 元素 | 详情 | 更换频率 |
|------|------|----------|
| **头像** | 640×640px，圆形裁切 | 按需 |
| **名称** | 25 字符 | 一次 |
| **类别** | 从预设列表选（无 "Furniture Sourcing"，最近是 "Shopping & Retail" 或 "Business Service"） | 一次 |
| **简介** | 139 字符，文字 | 随时 |
| **营业时间** | 按天设置 | 按需 |

### 内置工具（全部可选配）

| 工具 | 功能 | 是否推荐 |
|------|------|----------|
| **目录** | 添加产品/服务，每个有图 + 名称 + 价格（可设"询价"）/ 描述 + 链接 + 产品代码 | ✅ 核心！放 3 品类的代表产品 |
| **快捷回复** | 预设消息模板，`/` 快捷键触发 | ✅ 高频问题自动回复 |
| **自动回复** | 离线时自动发送消息 | ✅ "We'll respond within 24h" |
| **欢迎消息** | 新客户首次聊天自动发送 | ✅ 品牌问候 + CTA |
| **标签** | 给客户打标签分类 | ✅ "询盘/已报价/已成交" |
| **多设备** | 最多 4 台设备同时在线 | ✅ 老板 + 撒旦 |

### 目录装修要点

WA 目录是唯一有「设计感」的部分：

```
每个产品卡包含：
  - 图片（高 1:1 正方形）
  - 产品名称
  - 价格（选 "询价" 而非填数字）
  - 描述（简述 + SKU）
  - 链接（指向网站对应产品页）

建议放 9-12 个代表产品（床×4 / 沙发×4 / 餐桌×4）
```

### 限制

- 没有视频
- 没有富文本排版
- 没有自定义背景色
- 目录最多 500 个产品
- 简介不能放链接（可点击的只有 WA 系统自带链接）

## 四、Instagram 品牌号 ⭐

**装修=零。Instagram 不提供任何页面内自定义排版。所谓装修就是 Bio + Highlights。**

### 唯一可控元素

| 元素 | 详情 | 更换频率 |
|------|------|----------|
| **头像** | 110×110px（显示），建议上传 600×600 | 按需 |
| **Bio** | 150 字符 + 1 个可点击链接 + 最多 5 个快捷联系方式 | 每月 |
| **Name** | 30 字符（搜索权重高，可加关键词） | 按需 |
| **Highlights 封面** | 自定义图标，黑底 + 金线图案 | 按月 |
| **Highlights 内容** | 精选 Story 合集，无数量限制 | 每周 |
| **固定帖** | 最多 3 条置顶在网格最上方 | 按需 |

### Highlights 是 IG 版「装修」

这是 IG 唯一能做出「品牌页面感」的地方：

| Highlight 合集 | 建议内容 |
|----------------|----------|
| 🏭 **Factory** | 工厂实拍 Story · QC 过程 · 车间日常 |
| ✅ **Quality** | 质检报告 · 拒收案例 · 标准对比 |
| 👥 **Client** | 客户反馈 Story · 出货花絮 · 到货拆箱 |
| ℹ️ **About** | 品牌介绍 · 团队成员 · 20 年历史 |

### 限制

- 没有封面图
- 没有视频背景
- 不能排版
- Bio 不能富文本
- 所有内容都在时间线网格里，没有独立「主页设计」

## 五、总结 & 行动优先级

### 装修投入产出比

| 平台 | 装修复杂度 | 商业价值 | 推荐投入 |
|------|-----------|---------|----------|
| **Facebook** | 1 小时（配置多但简单） | 中高 | 🔴 优先 |
| **LinkedIn** | 0.5 小时（模板化，填内容即可） | 高（B2B 核心） | 🔴 优先 |
| **WA Business** | 1.5 小时（目录需要作图） | 高（转化终端） | 🟡 其次 |
| **Instagram** | 0.5 小时（Bio + Highlights 封面） | 中（引流为主） | 🟡 其次 |

### 一句话总结

- **FB**：能装得最像「品牌官网」，标签页+服务+评价全开
- **LI**：B2B 信任背书，模块固定但内容完全在手
- **WA**：不能装修但能干活——目录就是你的微网站
- **IG**：别想着装修了，把 Bio 写好 + Highlights 做好就完了
