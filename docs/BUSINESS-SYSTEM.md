## 四、内容协同引擎

### 4.1 核心原则

**「一个素材，多渠道分发，各自适配格式」**

不是「同一个内容发到所有平台」，而是「同一个主题，不同平台讲不同的话」。

### 4.2 内容生产管线

```
                    ┌─────────────┐
                    │  内容主题池   │ ← 路西法维护
                    │ （10+ 主题待命）│
                    └──────┬──────┘
                           │ 每周选 1 个主题
                           ▼
              ┌────────────────────────┐
              │     主题内容包生产       │
              │  路西法写母稿（长文）    │
              └───────────┬────────────┘
                          │
        ┌─────────────────┼─────────────────┐
        ▼                 ▼                   ▼
   ┌─────────┐      ┌─────────┐        ┌─────────┐
   │ 网站版本 │      │ IG 版本  │        │ LI 版本  │
   │ SEO 长文 │      │ Reels+帖 │        │ 专业文章 │
   │ 撒旦发布 │      │ 撒旦发布 │        │ 撒旦发布 │
   └─────────┘      └─────────┘        └─────────┘
        │                 │                   │
        └─────────────────┼───────────────────┘
                          │
                          ▼
              ┌────────────────────────┐
              │   玛门追踪数据 + SEO 效果 │
              │   米迦勒定期审计          │
              └────────────────────────┘
```

### 4.3 内容主题池（路西法管理）

| 类别 | 主题示例 | 适用平台 |
|------|----------|----------|
| **行业黑幕** | "为什么你的中国采购成本高了 30%" | IG + LI + 网站 |
| **工厂实况** | "今天我们拒收了什么" | IG + FB |
| **案例拆解** | "巴黎酒店如何省下 €15k 采购费" | LI + 网站 |
| **买手指南** | "第一次从中国采购的 7 个坑" | 网站 + LI |
| **趋势报告** | "2026 欧洲家具市场 3 个变化" | LI + 网站 + FB |
| **产品深挖** | "一张好床的 12 道工序" | IG + 网站 |

### 4.4 平台间内容联动规则

| 触发 | 联动动作 |
|------|----------|
| 网站发布新 Insight 文章 | IG Story 预告 → LI 发摘要 + 链接 → FB 群组讨论 |
| IG Reels 爆款（Reach > 5000） | 扩写成网站 Insight 文章 → LI 发深度版 → SEO 加内部链接 |
| LI 文章获高互动 | 提炼金句做 IG Feed 图 → FB 转发 |
| 客户成交案例 | 脱敏后 → 网站案例页 → LI 案例帖 → IG «客户说» Reel |
| WA 高频问题 | 整理成网站 FAQ / Insight 文章 → 社媒分发 |

## 五、SEO 持续优化引擎

### 5.1 技术 SEO（撒旦 + 米迦勒）

| 项目 | 当前状态 | 优化方向 |
|------|----------|----------|
| **Sitemap** | ✅ 已有 | 每次新增页面后更新 |
| **Canonical URL** | ✅ 已有 | 保持 |
| **Hreflang** | ✅ en/fr/de/it/es/pt | 视客户来源增语种 |
| **OG/Twitter Card** | ✅ 已有 | 每页独立 OG 图片 |
| **Core Web Vitals** | 待测 | 压缩图片 · 延迟加载 · CDN |
| **结构化数据** | ❌ 缺失 | 加 Organization + Article + Product Schema |
| **404 页面** | ❌ 缺失 | 自定义 404 引流回首页 |
| **robots.txt** | ❌ 缺失 | 添加 |

### 5.2 内容 SEO（路西法 + 玛门）

| 策略 | 执行 |
|------|------|
| **长尾关键词布局** | 每篇 Insight 文章主攻 1 个长尾词（如 "furniture sourcing from China for hotels"） |
| **内部链接网** | 每篇文章至少链向 2 篇其他相关内容 + 1 个产品板块 |
| **外部 Backlink** | LinkedIn 文章 + Facebook 群组 + 行业论坛引导回链 |
| **Google My Business** | 注册并关联网站（即使用不上本地流量，也加域名权威） |
| **定期内容更新** | 每月 review 并更新旧文章日期/数据，保持 freshness signal |

### 5.3 SEO KPI

| 指标 | Month 1 | Month 3 | Month 6 |
|------|---------|---------|---------|
| 收录页面数 | >10 | >30 | >50 |
| 日均自然搜索点击 | 5 | 30 | 100 |
| 目标关键词排名（前 10） | 0 | 3 | 10 |
| 网站 Domain Rating | — | >20 | >35 |

## 六、客户转化漏斗

```
  TOFU (Top of Funnel)          MOFU (Middle)            BOFU (Bottom)
  ┌─────────────────┐      ┌─────────────────┐      ┌─────────────────┐
  │ IG Reels 破圈    │      │ 网站 Insight 文章 │      │ WA Business 询盘 │
  │ LinkedIn 帖子    │  →   │ LI 深度内容      │  →   │ 邮件报价         │
  │ Google 搜索      │      │ Case Study       │      │ 视频通话         │
  │ FB 群组          │      │ 邮件 nurturing   │      │ 签单 · 成交      │
  └─────────────────┘      └─────────────────┘      └─────────────────┘
       发现                     考虑                      决策
```

### 转化路径设计

| 入口 | → 中转 | → 转化 |
|------|--------|--------|
| IG Reel | → 个人主页链接 → 网站 Insight 文章 | → WA 按钮 → 询盘 |
| Google 搜索 "China furniture sourcing" | → 网站 Insight 文章 | → WA/邮件 CTA |
| LinkedIn 长文 | → 网站 Case Study | → 联系表单 |
| FB 群组讨论 | → 网站首页 | → WA 按钮 |
| 直接访问 kalistorik.com | → 首页 CTA | → WA / 邮件 |

**关键设计**：每一个内容触点都必须有「下一步」——不能让流量断在任何一步。
