## 二、建议安装（按优先级）

### ⭐ 第一优先——装了就立刻能用

| 工具 | 用途 | 安装方式 | 免费？ |
|------|------|------|:--:|
| **ImageOptim** | 图片压缩（PNG/JPG），社媒图减小体积 | `brew install imageoptim` | ✅ |
| **Audacity** | 音频剪辑——裁剪/调音量/淡入淡出 | audacityteam.org | ✅ |
| **GIMP** | Photoshop 替代，修图、抠图、调色 | gimp.org | ✅ |

### ⭐⭐ 第二优先——提升社媒出图效率

| 工具 | 用途 | 安装方式 | 免费？ |
|------|------|------|:--:|
| **Canva** | 社媒模板/海报/品牌套件，一键套版 | canva.cn 下 Mac 版 | ✅（付费解锁更多） |
| **Figma** | UI/社媒设计、品牌视觉管理 | figma.com | ✅ |
| **OBS Studio** | 录屏/直播/多场景切换 | obsproject.com | ✅ |

### ⭐⭐⭐ 第三优先——锦上添花

| 工具 | 用途 | 安装方式 | 免费？ |
|------|------|------|:--:|
| **HandBrake** | 视频格式转换/压缩 | handbrake.fr | ✅ |
| **Inkscape** | SVG矢量编辑（Logo微调） | inkscape.org | ✅ |

---

## 三、操控能力对照

| 操控方式 | 能做什么 | 局限性 |
|------|------|------|
| **CLI 命令** | FFmpeg、yt-dlp 等——全自动 | 只有命令行工具能用 |
| **AppleScript** | 打开APP、按键、读窗口名 | UI元素暴露不全 |
| **cliclick** | 截图→分析→点坐标 | 坐标依赖屏幕分辨率 |
| **HTML管线** | 动画→Playwright录屏 | 我们的主力方案 |

---

## 四、建议安装顺序

```bash
# 一口气装完第一优先
brew install --cask imageoptim audacity gimp
```

Canva 和 Figma 去官网下 Mac 版，装完告诉我。
