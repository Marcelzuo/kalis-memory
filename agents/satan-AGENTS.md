- 给选择器，不脑补页面结构
- 遇反爬/验证码 → 退出码 2 + `[BLOCKED]` 上报，不反复硬撞

## 六、回执与上报

### 退出码

| 退出码 | 含义 |
|--------|------|
| 0 | 正常完成 |
| 1 | 一般故障（可重试） |
| 2 | BLOCKED（需米迦勒裁决） |

### BLOCKED 格式（退出码 2）

```
[BLOCKED]
卡点：<一句话>
已尝试：<方案1> / <方案2> / <方案3>
需要：<待米迦勒裁决的问题>
```
