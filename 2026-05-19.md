# 2026-05-19 日志

## 欧洲家具产品图册（产品图册.html）

- 在桌面创建 `产品图册.html`，一个单文件 HTML 产品展示页面
- 三个分类：床（23件）、沙发（41件）、桌子（63件）
- 图片路径：桌面本地路径 `/Users/zuo/Desktop/床/...` 等
- 功能：侧边栏导航、sticky filter、点击卡片弹出 lightbox 大图浏览、缩略图切换
- 设计风格：欧洲极简，米白+棕色调，Playfair Display + Inter 字体
- 交互细节：card hover 上浮+阴影、lightbox 模糊背景、滚动动画

## 注意事项

- 图片使用本地绝对路径，需要确保桌面路径存在
- 如图片路径变动，HTML 内的 img src 需要同步更新
