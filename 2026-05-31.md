# 日志 2026-05-31（续）

## KALIS TORIK 圆形头像终版定稿 🎉

### 最终方案
- **方法**：Chrome headless + Playwright 渲染，macOS 系统字体（San Francisco）
- **规格**：600×600 圆形 PNG，四角透明
- **输出**：桌面 `KALIS_TORIK_avatar_dark.png`（黑底白字）和 `KALIS_TORIK_avatar_light.png`（白底黑字+边框）

### 为什么之前都失败了
1. Pillow/FreeType 渲染小字时抗锯齿吃掉笔画 → 文字模糊
2. Pillow `draw.text()` y 坐标是 top-of-text 非 baseline → TORIK 叠在 KALIS 上
3. Google Fonts @import 在 headless Chrome 中加载不稳定 → 字体降级
4. Montserrat ExtraLight 200 太细 → 笔画被抗锯齿稀释
5. 120px 画布上 10px 字是物理极限 → 改用 600×600

### 最终参数
- 深色：黑底 #000000，字白 #FFFFFF，金线 #C9A96E 在上
- 浅色：白底 #FFFFFF + 边框 5px #E9E4DC，字黑 #000000
- KALIS：85px / LS 15px
- TORIK：50px / LS 25px
- 金线：70×10px
- 间距：金→KALIS 30px，KALIS→TORIK 60px
