## 方案撰写铁律（所有社媒项目通用）

### 一、研究原则
- 平台硬规格（图片尺寸、视频时长、字数限制、功能限制）**必须引用官方帮助中心原文**，不得推测
- 必须真实浏览平台页面（opencli browser 或官方文档），**不靠想当然**
- 不确定的数字宁可写区间也不编精确值

### 二、内容策略
- **拒绝传统配比思维**（如"产品 40% + 知识 30%"），这种打法在社媒上必死
- 必须基于**社交媒体行为心理学**设计内容框架——每个框架有明确的：
  - 心理触发机制（认知失调 / 窥探欲 / FOMO / 脆弱性=可信度 / 打破潜规则）
  - 3 秒 Hook（前 3 秒必须让人停下滑动的手指）
  - 具体帖文脚本（不是概念，是拿来就能拍的东西）
- 内容形式、表达方式 > 内容本身
- 剑走偏锋，出其不意，博眼球

### 三、方案结构（必须包含以下章节）
1. 老板速决（多方案选择 + 决策清单）
2. 平台硬规格（引用官方来源）
3. 内容哲学（心理触发器 + 爆火框架 + 内容模板）
4. 独有资产（USP + 客户痛点 + 真实/叙事人物线）
5. 执行日历（按周、分阶段、有漏斗逻辑 SHOCK→TRUST→CONVERT）
6. 视觉与表达规则
7. 安全合规（防封号铁律：官方政策引用 + 操作上限 + 新号热身期）
8. 自动化工具命令（如有）
9. KPI 追踪（周级 + 关键决策节点）

### 四、执行分工（默认模式，各项目可微调）
| 区 | 谁 | 做什么 |
|----|-----|--------|
| 🖌️ 内容制作 | 地狱三人组 | 路西法脚本 → 玛门视觉 → 撒旦初审 |
| 🚀 发布 + 🔍 互动 + 📊 数据 | 米迦勒 | 发帖、跟帖、DM、KPI |
| ✅ 终审 | 撒旦 + 米迦勒双签 → 老板拍板 | — |

### 五、工作模式
- **事件驱动，不设死时间**。老板上线叫谁谁开工
- 老板："路西法，出脚本"→ 动；"双签"→ 撒旦+米迦勒审；拍板 → 米迦勒执行

## KALIS TORIK 品牌全局约束（所有项目共用）

### 品牌定位
- European Furniture Sourcing Partner
- 20 years · 100+ vetted factories in China
- 核心站点：kalis-torik.com

### 战略定调（老板 2026-06-11）
- **不以赚钱为第一要务**——以增长和市场份额优先
- 未来采用**国外代理制**，突破传统贸易
- 以价换量，做突破性增长
- 内容不塑造"高价高端"，塑造"专业可信、量大价优"

### 内容禁区（写了就出事）
- ❌ 不提价格
- ❌ 不提付款方式
- ❌ 不提报价时间
- ❌ 不提看厂
- ❌ 不暴露工厂名称 / 位置 / 全貌

### VI 铁律
- 黑 #000000 / 白 #FFFFFF / 金 #C9A96E / 灰 #888888
- 禁止 #1a1a1a #0d0d0d #333
- 字体：Montserrat，不换字体、不压字距、不加粗变形
- Logo 结构：**金线在最上面** → KALIS → TORIK

### 表达风格
- 英文为主（目标客户欧洲 B2B）
- 不用公司腔，用酒吧聊天腔
- 用 "you" 不用 "client"
- 每帖只讲一个核心信息
- 数字保持真实感，不说"最好的""最低价"
