## 三、内容哲学 — FB B2B 特化版 [米]

### 3.1 为什么 IG 打法搬到 FB 必死

| 维度 | IG | FB |
|------|-----|-----|
| 注意力 | 3 秒抢停 | **3 行抢停**（首段说清"对我生意有什么用"） |
| 信任路径 | 视觉 → 好奇 → DM | **信息密度 → 验证 → Messenger** |
| 内容寿命 | 48 小时 | **2-4 周**（搜索 + Group 沉淀 + Google 索引 🔥） |
| 转化逻辑 | 看到 → 记住 → 某天 → Bio Link | **搜到 → 读 3-5 篇 → 看网站 → Messenger** |
| 失败原因 | Hook 不够炸 | **信息没有"可引用价值"** — 买家不截图发给老板 |

## 七、FB Group 渗透策略 [米]

### 7.1 目标 Group 类型

| 类型 | 关键词 | 优先级 |
|------|--------|--------|
| 家具进口商 | "Furniture Importers Europe" | 🔴 最高 |
| B2B 贸易 | "EU Importers" "B2B Sourcing China" | 🟡 高 |
| 电商零售 | "E-commerce Sellers Europe" | 🟢 中 |
| 区域商贸 | "Spain Import Export" "Germany Business" | 🟢 中 |

### 7.2 三阶段渗透

```
W1-W4 潜水 → 加入 5-7 个 Group，不说不推销
W5-W8 评论 → 回复行业信息（非推销），用个人号，不挂链接
W9+   发帖 → 每周 2 条讨论帖，价值 ≥ 70%，推销 ≤ 30%
```

### 7.3 安全铁律
- 不群发相同内容到多个 Group
- 不发帖间隔 < 48h
- 不用 Page 身份发广告
- 被踢 → 冷静 2 周

- ❌ "Best price" "Cheapest" "Lowest" "Guaranteed"
- ❌ 评论 "DM me" "Check my page"
- ❌ 自动化批量操作
- ❌ 短时间大量操作触发 spam

### 8.4 Messenger 三步法 [米]

```
Step 1: 互动预热（Group 评论 / Page 点赞）
Step 2: 加好友（附言说明你是谁）
Step 3: Messenger（对方认识你后才发）

冷 Messenger = spam 举报 = 限制
```

```
→ Monthly freight update: real rates, no fluff

Reply 'PDF' or 'FREIGHT'. No pitch. Just useful stuff.

KALIS TORIK · kalistorik.com
```

**铁律**：提及对方具体内容 · 首条不推销 · 回复手写 · ≤ 10 条/天 · 14 天未回不追
